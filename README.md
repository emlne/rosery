# Rosery Apart Website

Official website developed for Rosery Apart.  
https://rosery-apart.vercel.app/

## 🌿 About The Project

Rosery Apart website is a modern and responsive web application built to present the apartment's features, gallery, and contact information in a clean and user-friendly interface.

## 🚀 Built With

- React
- Vite
- CSS

## ✨ Features

- Responsive design
- Image gallery with slider
- Apartment details section
- Amenities section
- Contact information
- Clean and modern UI
